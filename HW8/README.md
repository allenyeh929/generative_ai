### 主題
* 第一階段 : 讓 LLM 從三個角色視角思考
  * 如果我是偵探（訊息接收者），我會怎麼想？
  * 如果我是旁觀者（無利益關係者），我會怎麼看？
  * 如果是兇手，背後的心理動機可能是什麼？
* 第二階段 : 根據第一階段的資訊，整合分析出不同視角的觀點，並娓娓道出不尋常的結尾。
